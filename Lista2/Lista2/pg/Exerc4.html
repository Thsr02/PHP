Crie um login

Exercício 4. Crie um programa que entre com a senha e o nome do usuário. Se senha = ”cotec” e o usuário =”unesp” emitir uma mensagem de “Seja Bem vindo ao Sistema de ativação.”, senão emitir mensagem de “Usuário ou senha inválidos!!, tente novamente!!!”. NOTA: Quando você quiser uma caixa de texto para entrada de informações secretas, utilize a tag < input type="password" > no lugar de < input type="text" >.
