Conversor de temperatura

Exercício 7. Elabore um script que receba de um formulário uma temperatura em graus Fahrenheit(F) e monte uma página dinâmica com a temperatura transformada para graus Celsius(C). O script deverá fazer esta mesma transformação para os próximos 10 valores sucessivos de F e montar uma tabela com os valores de F e C. Use a fórmula C = 5/9 * (F –32).
